# ExpenseTracker WebApp

This directory contains the Telegram WebApp files for ExpenseTracker bot.

## Files

- `datepicker.html` - Date picker WebApp for selecting expense dates

## GitHub Pages

This directory is served via GitHub Pages at:
`https://giovannimartelli.github.io/ExpenseTracker/datepicker.html`

## Usage

The date picker WebApp provides:
- Native HTML5 date input
- Quick selection buttons (Today, Yesterday, 7 days ago, 30 days ago)
- Validation to prevent future dates
- Telegram theme integration
- Mobile-optimized UI

The selected date is sent back to the bot in ISO format (YYYY-MM-DD).
